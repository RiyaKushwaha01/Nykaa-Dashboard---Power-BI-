📊 Nykaa Marketing Campaign Performance Dashboard | Power BI
📌 Project Overview

This project presents an interactive Power BI dashboard built using a marketing campaign dataset inspired by Nykaa's digital marketing activities. The dashboard provides comprehensive insights into campaign performance, customer engagement, revenue generation, and return on investment (ROI), enabling data-driven decision-making.

The project demonstrates how Power BI can transform raw marketing data into meaningful visualizations and actionable business insights.

🎯 Objectives
Analyze the effectiveness of marketing campaigns.

Monitor key performance indicators (KPIs).

Compare campaign performance across different channels and customer segments.

Identify high-performing campaigns and optimization opportunities.

Build an interactive business intelligence dashboard for marketing analytics.
