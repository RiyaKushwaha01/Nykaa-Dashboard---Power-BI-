# Netflix-Dashboard - Power-BI

This project visualizes Netflix’s global catalog of movies and TV shows using Power BI ,SQL Server and Excel based on the dataset from Kaggle: Netflix Movies and TV Shows Dataset. The goal was to build an interactive analytics dashboard for exploring Netflix content distribution by type, rating, genre, and country.

**🛠️Tools & Technologies Used** 

*Data Source: Kaggle

*ETL: Microsoft Excel

*Database: SQL server

*Visualization: Power BI and DAX

*Language: SQL


**Insights**

* Majority of content on Netflix is Movies (70%+).

* TV-MA and TV-14 dominate rating categories.

* Top genres include International Movies, Dramas, and Comedies.

* Global spread shows Netflix’s strong presence in North America, Europe, and Asia.
